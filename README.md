# Data Engineering Zoomcamp 2026

This repository contains coursework and practical exercises from the Data Engineering Zoomcamp (2026 cohort). It covers fundamental and advanced data engineering concepts through hands-on projects, including data ingestion pipelines, workflow orchestration with Prefect, cloud infrastructure with Terraform, and data warehousing.

## Structure

- **Week-1**: Data ingestion fundamentals with Docker, Postgres, and local pipeline development
- **Week-2**: Workflow orchestration using Prefect flows and cloud integration
